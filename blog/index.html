1. Definiciòn

Definiéndolo de forma sencilla, "HTML es lo que se utiliza para crear todas las páginas web de Internet". Más concretamente, HTML es el lenguaje con el que se "escriben" la mayoría de páginas web.

XHTML (Lenguaje de Marcado de Hipertexto Extensible) es una versión más estricta y limpia de HTML , que nace precisamente con el objetivo de remplazar a HTML ante su limitación de uso con las cada vez más abundantes herramientas basadas en XML . XHTML extiende HTML 4.0 combinando la sintaxis de HTML, diseñado para mostrar datos, con la de XML, diseñado para describir los datos.

Aunque HTML es un lenguaje que utilizan los ordenadores y los programas de diseño, es muy fácil de aprender y escribir por parte de las personas. En realidad, HTML son las siglas de HyperText Markup Language

XHTML es una adaptación de HTML 4.01 al lenguaje XML, por lo que mantiene casi todas sus etiquetas y características, pero añade algunas restricciones y elementos propios de XML. La versión XHTML 1.1 ya ha sido publicada en forma de borrador y pretende modularizar XHTML. También ha sido publicado el borrador de XHTML 2.0, que supondrá un cambio muy importante respecto de las anteriores versiones de XHTML.

Las páginas y documentos creados con XHTML son muy similares a las páginas y documentos HTML. Las discusiones sobre si HTML es mejor que XHTML o viceversa son recurrentes en el ámbito de la creación de contenidos web, aunque no existe una conclusión ampliamente aceptada.

El lenguaje XHTML es muy similar al lenguaje HTML. De hecho, XHTML no es más que una adaptación de HTML al lenguaje XML. Técnicamente, HTML es descendiente directo del lenguaje SGML, mientras que XHTML lo es del XML (que a su vez, también es descendiente de SGML).

Sintaxis XHTML

Sintaxis XHTML es muy similar al HTML, la sintaxis y casi todos los elementos HTML en XHTML válido. Pero cuando se escribe un documento XHTML tienes que poner un poco más de atención para hacer que su documento HTML a XHTML

Declaración DOCTYPE

Todos los documentos XHTML deben tener una declaración DOCTYPE al principio. Hay tres tipos de DOCTYPE declaraciones, que se describen detalladamente en XHTML Doctypes capítulo. Este es un ejemplo de uso DOCTYPE: 

                
// SINTAXIS
!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" 
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Mayúsculas y minúsculas

XHTML es sensible a mayúsculas y minúsculas lenguaje de marcado. Todas las etiquetas y atributos XHTML deben estar escritas sólo en minúsculas. 
                
// SINTAXIS

A Href="/xhtml/xhtml_tutorial.html">XHTML Tutorial


a href="/xhtml/xhtml_tutorial.html">XHTML Tutorial>

Mínimo UN documento XHTML

Ejemplo siguiente se muestra un contenido mínimo de un documento XHTML 1.0. 
                
// SINTAXIS
?xml version="1.0" encoding="UTF-8"?>
< !DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
html xmlns="http://www.w3.org/TR/xhtml1" xml:lang="en" lang="en">

   head>
      < title>Every document must have a title
   < /head>

   body>
   ...your content goes here...
  < /body>
/html>
